# 叙事与战斗输出

## 回合输出顺序

战斗和紧张场景按以下顺序输出：

1. 行动经济：PC、可控单位、自主 NPC、敌方分别已用/可用的动作、附赠动作、移动、反应、资源。
2. 自主意图：主要 NPC/敌人的意图、理由、可能偏离。
3. 骰子结算：公式、骰面、加值、DC/AC、结果、伤害、状态。
4. 场景叙事：2-5 句具体画面，包含位置、环境、伤口、证据、机关、局势变化。
5. 战斗台词：1-3 句短台词，可包含威胁、嘲讽、队友建议、求救、反制宣言。
6. 状态更新：HP、法术位、资源、关系、时钟、地图变化。
7. 下一步选项：2-4 个方向，每个标注检定、消耗、风险和 NPC 自主倾向。

## 台词规则

台词要服务局势，不要盖过玩家选择：

- 敌人台词可以威胁、拖延、交易、扰乱、恐吓人质、泄露半真半假的线索。
- 队友台词可以建议、提醒风险、质疑命令、请求兑现承诺、催促撤退。
- 被救援对象可以恐惧、求救、误导或提供现场线索。
- 台词不能泄露 NPC 不该主动泄露的秘密。
- 台词暗示可行动线索时，下一组选项必须把它转成可检定或可选择的行动。

## 行动选项示例

```text
1. 用火焰箭点燃油桶，逼灰袍人离开账本
   检定：法术攻击 +5 vs 油桶 AC10；命中后油桶爆燃，附近生物敏捷豁免 vs DC13
   消耗：吴迪动作；1 个戏法，无法术位
   移动：无
   风险：失败会暴露你的施法位置；爆燃可能波及林微
   NPC 自主倾向：林微若判断会伤到人质，可能自主拖人后撤
```

```text
2. 命令林微拖走账本
   检定：无直接检定；先做林微自主意图判定
   消耗：吴迪免费短口令；林微若接受，消耗林微动作和移动
   风险：林微可能因恐惧、证据优先或敌方威胁而变体执行
   NPC 自主倾向：trust 高倾向协助；fear 高倾向先救人或撤离
```

## 多单位提醒

当玩家控制多个单位或有召唤物时，先给清单：

```text
可控单位：
- 吴迪：动作可用，附赠动作可用，移动 9m，反应可用，法术位 1环 1/2。
- 法师之手：无独立攻击；可移动 9m；若要精细操控需吴迪动作。
- 召唤物：独立回合，动作可用，移动 9m，反应可用；听从上个命令。

自主单位：
- 林微：自主 NPC，不可直接控制；可被请求、说服或命令。
- 灰袍人：重要敌人，自主行动。
```

## 好玩的文字交互

- 每个检定都改变局势。失败也给信息、代价或新入口。
- 每张地图至少同时推动两个阵营目标。
- 每场战斗至少有一个环境按钮。
- 每个主要 NPC 至少有一个想让玩家帮忙的事，和一个不愿承认的事。
- 长休安排营地对话、梦境、关系推进或敌方时钟。
- 野外探索让玩家在速度、安全、线索、资源之间取舍。
