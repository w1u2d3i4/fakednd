# Sources

This framework uses external sources as structural anchors, not as copied rules text.

- D&D Beyond Free Rules, Creating a Character: class/background/species/ability-score creation flow and standard array/point-buy anchors. https://www.dndbeyond.com/sources/dnd/free-rules/creating-a-character
- D&D Beyond Free Rules, Rules Glossary: DC, AC, saving throws, attack rolls, initiative and core d20 terminology anchors. https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary
- D&D Beyond Free Rules, Feats: origin feat category and feat-style design anchors. https://www.dndbeyond.com/sources/dnd/free-rules/feats
- BG3 Wiki, Classes: BG3 class, subclass, background and race lists, including optional post-launch subclass entries. https://bg3.wiki/wiki/Classes
- wh40k-test.xyz: reference for the repeated-choice personality-test interaction pattern used by the five life-experience attribute questions. https://wh40k-test.xyz/

When exact current BG3 spell, class, subclass, feat or patch behavior matters, verify against a current BG3 rules source before finalizing mechanics. In play, always present concise mechanics rather than reproducing long proprietary rules text.
